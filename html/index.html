HTML이란 Hypertext Markup Language의 약자로 웹을 이루는 가장 기초적인 구성 요소로, 웹의 구조를 잡는다. 즉 뼈대를 세운다고 할 수 있습니다. HTML을 이용해 구조를 잡아주어야 추후에 CSS(SCSS)를 이용해 Style을 잡아 브라우저를 예쁘게 꾸미거나, Javascript를 사용하여 동적인 구현을 할 수 있습니다.

HTML의 기본적인 개념과 구조

HTML의 시작하기위해 익혀두어야하는 기본적인 개념과 사용할 수 있는 구조를 알아가며 기본을 익힙니다. 차근차근 구조를 만드는 방법을 알아봅시다.

HTML의 구성요소(Element)

HTML의 기본구조(뼈대)를 세우기 위해 어떠한 요소들이 합쳐져 구성하는지 알 수 있습니다. 구성되는 요소들의 각각의 맡은 역할을 알아봅시다.

HTML의 display 속성

HTML의 display 속성 중 기본이 되는 블록 레벨요소(Block Level Element)와 인라인 레벨요소(Inline Level Element)의 차이점을 알아보자.

HTML의 시멘틱 요소(Semantic Element)

HTML의 시멘틱 요소(Semantic Element)를 알아보고, 시멘틱 마크업(Semantic Markup)을 할 수 있도록 공부하고, 시멘틱 요소(Semantic Element)를 맞는 자리에 작성할 수 있도록 학습합시다.

HTML의 요소(Element)의 종류

HTML의 주로 사용되는 요소(Element)를 알아보고, 어떤용도로 사용되는지, 어느곳에서 적재적소에 맞게 사용하는 것이 맞는지 학습해봅시다.

HTML의 임베디드 요소(Embedded Element)

HTML의 임베디드 요소(Embedded Element)를 알아보고, 각 요소들의 사용하는 용도를 알아보고, 알맞게 사용하는 방법을 알아봅시다.

HTML의 표를 만드는 태그

HTML의 표를 만들 수 있는 태그를 알아보고, 어느곳에서 주로 사용되는 확인하고 사용해봅시다.
