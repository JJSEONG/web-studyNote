Git의 설치와 설정하기

⭐️ Git 설치

Git은 Window 운영체제와 Mac 운영체제가 서로 설치하는 방법이 다르다. 설치하는 방법을 알아보고, 설치를 진행해보자.

⭐️ Window 운영체제 Git 설치

Git을 설치하기위해 https://git-scm.com/으로 접속하여 Download를 클릭하여 설치를 진행해 주면 된다. 현재 작성자는 Mac 운영체제를 사용하고 있어서 Mac용으로 출력되지만, Window 운영체제를 사용하시는 분들은 Window용으로 출력된다.

추후에 약관을 읽어보고 Next를 누르다 Install 창이 뜨면 Install 을 클릭하여 설치를 진행한다. 보통은 따로 변경해 줄 필요가 없기 때문에, Next만 쭉 누르고 설치를 진행해도 무관하다.

설치 완료 후 cmd 창을 열어 git --version 을 입력하면 현재 설치 된 Git의 버전을 나타내는 것을 볼 수 있다.

⭐️ Mac 운영체제 Git 설치

Terminal(터미널)을 열어 git --version 을 입력한다.

git --version 을 입력 후 다음과 같은 창이 나오면 설치를 눌러 진행한다.

설치 완료 후 cmd 창을 열어 git --version 을 입력하면 현재 설치 된 Git의 버전을 나타내는 것을 볼 수 있다.


위의 방법으로 설치가 되지 않는다면, Homebrew 라는 프로그램을 사용하여 설치를 진행한다. https://brew.sh/index_ko에 접속하여 Homebrew 설치하기 아래에 있는 명령어를 복사하여 Terminal(터미널)에서 붙여넣고 설치를 진행한다.


명령어 : & /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Homebrew 설치를 완료 하였으면 Terminal(터미널)에서 brew install git 명령어를 입력하여 Git 설치를 완료할 수 있다.


Window 운영체제와 마찬가지로 설치 완료 후 Terminal(터미널) 창을 열어 git --version 을 입력하면 현재 설치 된 Git의 버전을 나타내는 것을 볼 수 있다.

⭐️ Git 최초설정

Git을 설치 후 사용하기 위해, 사용자의 정보를 입력해 주어야한다.

git config --global user.email "사용자의 이메일"
git config --global user.name "사용자의 이름"

사용자 정보를 입력 후 git config --list 를 입력하면 사용자의 정보가 제대로 입력되었는지 확인 할 수 있다.
