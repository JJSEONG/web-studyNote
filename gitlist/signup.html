Github 시작하기

⭐️ Github 회원가입과 저장소(Repository)

Git의 사용자 설정까지 진행했으면, 이제 원격저장소(Remote Repository)인 Github의 회원가입을 진행하여 사용하기 전에 환경을 구축하고 새로운 저장소(Repository)를 생성하는 법을 알아보자.

⭐️ Github 회원가입

Github을 사용하기 전에 먼저 회원가입을 하여야한다. https://github.com/ 의 홈페이지로 이동하여 회원가입을 진행한다.

⭐️ Github 저장소(Repository) 생성하기

위의 과정에서 아이디를 생성했으면 로그인을 하고, Repositories 탭으로 이동하여 새로운 저장소(Repository)를 생성한다.
