HTML의 표를 만드는 태그

⭐️ HTML 문서에 표 만들기

HTML의 표를 생성하는 태그를 알아보려고한다. 표에는 행(row)와 열(column)으로 이루어져 있으며, 이 태그들을 어떻게 사용하는지 확인해보자.

⭐️ <table>, <tr>, <td>, <th> 태그

<table>, <tr>, <td>, <th> 태그(Tag)들이 모여서 표의 기본적인 구성을 갖춘다.

구성 요소 의미
<table> 데이터 표를 정의한다.
<tr> 테이블(표)의 한 줄(행)을 나타낸다.
<td> 테이블(표)의 한 칸(열, 셀)을 나타낸다.
<th> 테이블(표)의 제목이 되는 칸을 나타낸다.
HTML
<table>
<tr>
<td>첫번째줄 첫번째칸</td>
<td>첫번째줄 두번째칸</td>
<td>첫번째줄 세번째칸</td>
</tr>
<tr>
<td>두번째줄 첫번째칸</td>
<td>두번째줄 두번째칸</td>
<td>두번째줄 세번째칸</td>
</tr>
</table>

CSS
table { border-collapse : collapse; }
td { border : 1px solid white }

위와 같이 작성하면 아래와 같은 결과를 가지게된다.

첫째줄 첫째칸 첫째줄 둘째칸 첫째줄 셋째칸
둘째줄 첫째칸 둘째줄 둘째칸 둘째줄 셋째칸

⭐️ <caption>, <figcaption> 태그

<caption> 태그(Tag)는 표의 제목을 붙일 때 사용한다. <table> 태그 바로 아래에 사용되어야하며, 하나의 <table>에 하나의 <caption> 혹은 <figcaption> 만 사용이 가능하다.

태그명 의미
<caption> 표의 제목으로 사용되며, 제목이 상단 중앙에 표시된다.
<figcaption> 표의 제목으로 사용되며, 제목이 상단 왼쪽에 표시된다.
HTML
<table>
<caption>표의 구성요소</caption>
<tr>
<td>첫번째줄 첫번째칸</td>
<td>첫번째줄 두번째칸</td>
<td>첫번째줄 세번째칸</td>
</tr>
<tr>
<td>두번째줄 첫번째칸</td>
<td>두번째줄 두번째칸</td>
<td>두번째줄 세번째칸</td>
</tr>
</table>

CSS
table { border-collapse : collapse; }
td { border : 1px solid white }

위와 같이 작성하면 아래와 같은 결과를 가지게된다.

표의 구성요소
첫째줄 첫째칸 첫째줄 둘째칸 첫째줄 셋째칸

⭐️ colspan, rowspan 속성

이 두개의 속성은 표의 행이나 열을 합치는 속성이다 [병합]. 셀을 합치는 속성으로 셀을 만드는 <td> 태그 혹은 <th> 태그에서 사용한다.

속성명 의미
colspan 열을 병합하며, 지정한 숫자만큼 병합한다. colspan="합쳐지는 열의 개수"
rowspan 행을 병합하며, 지정한 숫자만큼 병합한다. colspan="합쳐지는 행의 개수"
HTML
<table>
<tr>
<td colspan="2">01</td>
</tr>
<tr>
<td rowspan="2">03</td>
<td>04</td>
</tr>
<tr>
<td>06</td>
</tr>
</table>

CSS
table { border-collapse : collapse; }
td { border : 1px solid white }

위와 같이 작성하면 아래와 같은 결과를 가지게된다.

01
03 04
06

⭐️ <thead>, <tbody>, <tfoot> 태그

위의 태그(Tag)들은 테이블의 내용을 그룹화하여 코드를 읽을 때 명확히 구별해주는 역할을 하며, 화면을 볼 수 없는 분들이 사용하는 스크린 리더를 통해 표의 구조를 쉽게 이해할 수 있다. 즉, 웹접근성이 향상된다.

속성명 의미
<thead> 표의 머리글(제목)을 지정한다. 테이블 태그 당 한번만 사용가능하다.
<tbody> 표의 본문을 지정한다.
<tfoot> 표의 바닥글(요약)을 지정한다. 테이블 태그 당 한번만 사용가능하다.
HTML
<table>
<thead>
<tr>
<th>thead 01</td>
<th>thead 02</td>
<th>thead 03</td>
</tr>
</thead>
<tbody>
<tr>
<th>tbody 01</td>
<th>tbody 02</td>
<th>tbody 03</td>
</tr>
</tbody>
<tfoot>
<tr>
<th>tfoot 01</td>
<th>tfoot 02</td>
<th>tfoot 03</td>
</tr>
</tfoot>
</table>

CSS
table { border-collapse : collapse; }
th,td { border : 1px solid white }

위와 같이 작성하면 아래와 같은 결과를 가지게된다.

thead 01 thead 02 thead 03
tbody 01 tbody 02 tbody 03
tfoot 01 tfoot 02 tfoot 03

⭐️ <colgroup>, <col> 태그

한 열에 있는 모든 셀에 같은 스타일을 적용하고자 할 때 사용한다. <col> 태그는 닫는 태그가 없기 때문에 <col> 태그로 둘 이상의 열을 묶어 같은 스타일을 저장하려면 span 속성을 이용해 <col span="2"> 형식으로 설정한다. <colgroup> 태그를 사용해 여러 열을 묶어 스타일을 적용하려면 <colgroup> 태그 안에 묶고 싶은 열의 <col> 태그를 작성하면 된다.

HTML
<table>
<colgroup>
<col span = "2" style = "background-color : red" >
</colgroup>
<tr>
<th>01-01</td>
<th>01-02</td>
<th>01-03</td>
</tr>
<tr>
<th>02-01</td>
<th>02-02</td>
<th>02-03</td>
</tr>
<tr>
<th>03-01</td>
<th>03-02</td>
<th>03-03</td>
</tr>
</table>

CSS
table { border-collapse : collapse; }
th,td { border : 1px solid white }

위와 같이 작성하면 아래와 같은 결과를 가지게된다.

01-01 01-02 01-03
02-01 02-02 02-03
03-01 03-02 03-03
