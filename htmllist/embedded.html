HTML의 임베디드 요소(Embedded Element)

⭐️ 임베디드 요소(Embedded Element)

HTML의 임베디드 요소(Embedded Element)는 다른 콘텐츠를 현재 문서에 가져와서 삽입하는 태그이다. 주로 이미지나 비디오 등을 삽입할 때 사용하며, 그 태그들의 종류와 속성을 알아보자.

⭐️ <img /> 태그
Inline Level Element

<img /> 태그(Tag)는 빈 요소(Empty Element)로 사용되며, 원하는 이미지를 불러와 현재 문서에 삽입할 때 사용한다. 속성과 속성값을 같이 사용한다.

속성명 의미
src [필수] 이미지의 경로를 작성하며, src="이미지 경로" 로 사용한다.
alt [필수] 이미지가 보이지 않을 때 이미지를 설명해주는 대체 텍스트이며, alt="설명" 으로 사용한다.
width 이미지의 넓이를 지정하며, width="크기" 로 사용한다.
height 이미지의 높이를 지정하며, height="크기" 로 사용한다.
<img src = "이미지 경로" alt = "이미지 설명" width = "이미지 넓이 지정" height = "이미지 높이 지정">

⭐️ <audio> 태그
Inline Level Element

<audio> 태그(Tag)는 원하는 오디오를 가져와 삽입할 때 사용한다. 여러 속성을 가지고있으며, 속성과 속성값을 같이 사용한다.

속성명 의미
src [필수] 오디오의 경로를 작성하며, src="오디오 경로" 로 사용한다.
controls 오디오를 컨트롤 할 수 있는 제어창이 같이 생성된다.
autoplay 브라우저를 실행하면 오디오를 자동 실행한다.
단, 크롬에서 autoplay는 muted와 함께 사용하여야 적용된다.
loop 오디오 재생이 끝이나면 다시 반복하여 재생하여준다.
muted 브라우저에서 오디오의 소리를 음소거하여 재생한다.
preload 브라우저에게 오디오 파일도 같이 로드 되어야하는지 저자가 생각하는 최적의 사용자 경험은 무엇인지 힌트를 준다.
단, autoplay 속성이 지정되어 있으면 preload 속성은 무시된다.
  • auto : [기본값] 문서를 불러올 때, 오디오 파일을 같이 다운로드한다.
  • metadata : 오디오 파일 전체를 다운로드 하지 않고, 메타데이터(metadata)만 불러온다.
  • none : 오디오 파일을 다운로드 하지 않는다. 재생버튼을 눌러줘야 다운로드 시작한다.
<audio controls autoplay loop muted preload = "auto | metadata | none">
<source src = "오디오 경로">
</audio>

<source src = "오디오 경로"> 는 아래와 같이 <audio> 태그의 속성으로 사용이 가능하다.
<audio src = "오디오 경로"><audio>

⭐️ <video> 태그
Inline Level Element

<video> 태그(Tag)는 원하는 동영상을 가져와 삽입할 때 사용한다. 여러 속성을 가지고있으며, 속성과 속성값을 같이 사용한다.

속성명 의미
src [필수] 비디오의 경로를 작성하며, src="비디오 경로" 로 사용한다.
controls 동영상을 컨트롤 할 수 있는 제어창이 같이 생성된다.
autoplay 브라우저를 실행하면 비디오를 자동 실행한다.
단, 크롬에서 autoplay는 muted와 함께 사용하여야 적용된다.
loop 비디오 재생이 끝이나면 다시 반복하여 재생하여준다.
muted 브라우저에서 비디오의 소리를 음소거하여 재생한다.
poster 비디오를 재생하기 전, 썸네일을 표시한다. poster="썸네일 경로"
preload 브라우저에게 비디오 파일도 같이 로드 되어야하는지 저자가 생각하는 최적의 사용자 경험은 무엇인지 힌트를 준다.
단, autoplay 속성이 지정되어 있으면 preload 속성은 무시된다.
  • auto : [기본값] 문서를 불러올 때, 비디오 파일을 같이 다운로드한다.
  • metadata : 비디오 파일 전체를 다운로드 하지 않고, 메타데이터(metadata)만 불러온다.
  • none : 비디오 파일을 다운로드 하지 않는다. 재생버튼을 눌러줘야 다운로드 시작한다.
width 비디오의 넓이를 지정한다.
height 비디오의 높이를 지정한다.
<video controls autoplay loop muted poster = "썸네일 경로" preload = "auto | metadata | none" width = "이미지 넓이 지정" height = "이미지 높이 지정">
<source src = "비디오 경로">
</video>

⭐️ <iframe> 태그
Inline Level Element

<iframe> 태그(Tag)는 현재 HTML 문서에서 다른 HTML 문서 혹은 다른 사이트를 불러올 때 사용한다. 문서에서 너무 많은 콘텐츠의 양을 취급해야 할 때 주로 사용한다.

속성명 의미
src [필수] <iframe>의 경로를 작성하며, src="<iframe> 경로" 로 사용한다.
width <iframe>의 넓이를 지정한다.
height <iframe>의 높이를 지정한다.
name <iframe>의 이름을 지정한다.
<iframe> src = "<iframe> 경로" width = "<iframe> 넓이 지정" height = "<iframe> 높이 지정" </iframe>
