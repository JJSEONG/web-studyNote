HTML의 display 속성

⭐️ display 속성

HTML의 여러가지 display 속성 중 가장 기본으로 알아야하는 블록 레벨요소(Block Level Element)와 인라인 레벨요소(Inline Level Element)가 있다. 이 둘의 차이점을 확실히 알아두고, HTML 문서 작성 시 적재적소에 맞는 요소를 사용할 수 있도록 연습해야한다. 블록 레벨요소(Block Level Element)와 인라인 레벨요소(Inline Level Element)의 차이점을 알아보자.

⭐️ 블록 레벨요소(Block Level Element)

블록 레벨요소(Block Level Element)는 HTML의 요소들 중에 많은 비중을 차지하고있는 요소이다.
대표적인 예로는 <body>, <h1> ~ <h6>, <header>, <main> 등 시멘틱 태그(Semantic Tag)와 <div>, <p> 등 수많은 요소들이 포함되어있다.

⭐️ 블록 레벨요소(Block Level Element) 의 특징

  • 블록 레벨요소는 부모 요소의 전체를 차지하여 블록처럼 만든다.
  • 블록처럼 만들어지기 때문에, 작성한 순서대로 위에서 아래로 쌓인다.
  • 새로운 블록 레벨요소가 나오면, 새로운 줄을 생성하여 만들어진다.
  • width, height와 margin, padding 값을 모두 가질 수 있다.

<p>블록 레벨요소는 한 줄을 가득 채웁니다.</p>

⭐️ 인라인 레벨요소(Inline Level Element)

인라인 레벨요소(Inline Level Element)는 블록 레벨요소와 상반된 속성을 가진 요소이다.
대표적인 예로는 <span>, <img>, <a>, <em>, <i> 등의 요소들이 포함되어있다.

⭐️ 인라인 레벨요소(Inline Level Element) 의 특징

  • 인라인 레벨요소는 태그에 할당받은 크기만큼만 공간을 차지한다.
  • 할당받은 크기만큼만 차지하기 때문에, 옆으로 나열된다.
  • 태그가 없이 작성된 글자와 같은 취급이 된다.
  • width, height 값을 지정할 수 없다.
  • padding 은 상하좌우 값을 모두 가질 수 있다.
  • margin 은 left,right(좌우)값만 가질 수 있다.
HTML
<span>인라인 레벨요소는 컨텐츠만큼 공간을 가집니다.</span>
<span>인라인 레벨요소는 옆으로 배치됩니다.</span>

위의 내용에 css를 통하여 배경색을 채워보겠습니다.

CSS
span {
background-color: red;
}
인라인 레벨요소는 컨텐츠만큼 공간을 가집니다. 인라인 레벨요소는 옆으로 배치됩니다.

위의 예시와 같이 인라인 레벨요소는 할당된 컨텐츠만큼 공간을 가지고, 옆으로 배치된다.

CSS

}

⭐️ 인라인-블록 레벨요소(Inline-block Level Element)

인라인-블록 레벨요소(Inline-block Level Element)는 블록 레벨요소(Block Level Element)와 인라인 레벨요소(Inline Level Element)를 합쳐서 사용한다고 생각하면 좋다.
대표적인 예로는 <input>, <button>, <select>, <textarea> 등의 요소들이 포함되어있다.

⭐️ 인라인-블록 레벨요소(Inline-block Level Element) 의 특징

  • 블록 레벨요소에 inline-block 속성을 부여하면, 인라인 레벨요소와 같이 줄바꿈이 일어나지 않으며, 다른 요소들과 한 줄에 같이 작성이 된다.
  • 인라인 레벨요소에 inline-block 속성을 부여하면, 인라인 레벨요소에서 지정할 수 없었던 width, height 값을 지정할 수 있으며, margin, padding 모든 값을 지정할 수 있다.
HTML
<p>블록요소를 인라인-블록요소 설정1</p>
<p>블록요소를 인라인-블록요소 설정2</p>

위의 내용에 css를 통하여 inline-block으로 변경해보겠습니다.

CSS
p {
display: inline-block;
background-color: red;
}

블록요소를 인라인-블록요소 설정2

위의 내용과 같이 블록 레벨요소를 인라인-블록 요소로 변경하면, 더이상 줄바꿈이 일어나지 않고 옆으로 배치됩니다.


HTML
<span>인라인요소를 인라인-블록요소 설정1</span>
위의 내용에 css를 통하여 inline-block으로 변경 후 크기를 지정해보겠습니다.

CSS
span {
display: inline-block;
background-color: red;
width: 240px;
height: 50px;
}

위의 내용과 같이 인라인 레벨요소를 인라인-블록 요소로 변경하면, width와 height값으로 크기를 지정할 수 있습니다.


HTML
<span>인라인요소를 인라인-블록요소 설정2</span>
위의 내용에 css를 통하여 inline-block으로 변경 후 margin,padding값을 지정해보겠습니다.

CSS
span {
display: inline-block;
background-color: red;
margin: 20px;
padding: 10px;
}
인라인요소를 인라인-블록요소 설정2

위의 내용과 같이 인라인 레벨요소를 인라인-블록 요소로 변경하면, padding값은 그대로 모든값을 적용할 수 있으며, 좌우만 적용되던 margin값은 상하좌우 모두 적용되는 것을 알 수 있습니다.
