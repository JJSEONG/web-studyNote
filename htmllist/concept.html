HTML의 기본적인 개념과 구조

⭐️ 사전적 정의

하이퍼 텍스트 마크업 언어(영어: Hyper Text Markup Language, HTML, 문화어: 초본문표식달기언어, 하이퍼본문표식달기언어)는 웹 페이지를 위한 지배적인 마크업 언어다. 또한, HTML은 제목, 단락, 목록 등과 같은 본문을 위한 구조적 의미를 나타내는 것뿐만 아니라 링크, 인용과 그 밖의 항목으로 구조적 문서를 만들 수 있는 방법을 제공한다. 그리고 이미지와 객체를 내장하고 대화형 양식을 생성하는 데 사용될 수 있다. HTML은 웹 페이지 콘텐츠 안의 꺾쇠 괄호에 둘러싸인 "태그"로 되어있는 HTML 요소 형태로 작성한다. HTML은 웹 브라우저와 같은 HTML 처리 장치의 행동에 영향을 주는 자바스크립트와 본문과 그 밖의 항목의 외관과 배치를 정의하는 CSS 같은 스크립트를 포함하거나 불러올 수 있다. HTML과 CSS 표준의 공동 책임자인 W3C는 명확하고 표상적인 마크업을 위하여 CSS의 사용을 권장한다.

Web을 구성하는 기초가 되는 HTML(Hyper Text Markup Language)은 대표적인 마크업 언어이다. 마크업 언어(Markup Language)란 태그 등을 이용하여 문서나 데이터의 구조를 구성하는 언어이다. HTML은 WebSite의 뼈대와 같은 역활을 하며, 이 뼈대가 없으면 CSS(SCSS)를 이용해 스타일을 입힐 수 없으며, Javascript를 이용해 동적인 구현도 어려울 수 있다.

⭐️ HTML의 구조

HTML은 마크업 언어로 태그를 이용하여 WebSite를 구성한다.

Html html 태그를 사용하여 문서의 처음과 끝 즉, 문서의 범위를 나타낸다.
Head head 태그는 제목이나 검색키워드 등 문서의 정보를 가지는 태그이다.
body body 태그 안에 여러 태그들을 이용하여, 웹문서를 만들 때 body 태그 안에 작성한다.(문서의 구조)
<html> [html의 여는태그]
<head> [head의 여는태그]
<body> [body의 여는태그]
</body> [body의 닫는태그]
</head> [head의 닫는태그]
</html> [html의 닫는태그]

위의 내용과 같이 HTML의 기본적인 구조는 <여는태그>내용<닫는태그>로 구성이 되어진다.
