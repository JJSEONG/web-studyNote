HTML의 요소(Element)의 종류

⭐️ 요소(Element)의 종류

HTML의 요소(Element)는 수없이 많은 종류가 있습니다. 그 중에서도 HTML 문서에 주로 사용되며, 개인적으로 Web 개발을 할 때, 이정도는 최소한으로 알아야 한다고 생각하는 것 들을 정리하였기 때문에 부족할 수 있습니다. 더욱 더 자세한 내용을 알고 싶으시면, MDN 문서 등 공식적인 문서를 찾아보는 것도 좋은 방법입니다.

⭐️ !DOCTYPE 선언문

태그(Tag)를 선언하기 전에, <!DOCTYPE html> 을 선언하여 이 웹페이지가 HTML로 구성이 된다는 점을 웹페이지에 알려주는 역할입니다. 이 선언이 없이 HTML의 문서를 작성하면, 웹브라우저에서 표준 모드(Standards Mode)가 아닌 비표준 모드(Quirks Mode) 로 인식해 크로스 브라우징에 오류가 발생 할 수 있어, 개발환경 외의 다른 브라우저에서는 아무리 깔끔하고 멋있게 작성한 문서라도 원하지 않는 모습으로 표현될 수도 있다.

⭐️ <html> 태그

<html> 태그는 !DOCTYPE을 선언 한 후 html의 문서의 시작과 끝을 알려주는 태그이다.
<html> 태그의 속성으로는 lang 이라는 속성이 들어가며, 이는 language의 약자로, 웹브라우저의 표준 언어를 지정하는데 사용된다.
en = 영어 / ko = 한국어 / ja = 일본 등의 속성값이 존재한다.

⭐️ <head> 태그

<head> 태그는 <html> 태그 밑에 작성되며, 웹브라우저의 정보 이름이나 언어방식 등이 작성되어있다.
대표적으로 <meta> 태그와 <title> 태그가 있다.

<meta> 태그는 웹브라우저의 언어, 정보, 키워드 등을 작성할 수 있다.
<title> 웹브라우저의 제목을 작성할 수 있다.

⭐️ <link> 태그

빈 요소(Empty Element)이며, head 태그 안에 사용된다. 외부에서 문서를 가져올 때 사용하는 태그이다. 주로, css, javascript 같은 문서를 불러오는데 사용한다.

<link rel = 현재 문서와의 관계 href = 외부 문서의 링크(경로)>

rel 속성
: 불러 올 외부 문서와 현재 작성되고 있는 문서와의 관계를 나타낸다. 주로 사용되는 stylesheet 는 css 문서를 불러올 때 사용되는 속성값이고, icon은 favicon을 불러올 때 사용되는 속성값이다.
href 속성
: 불러 올 외부 문서의 링크(경로)를 작성하여 현재 작성되는 페이지에서 작성된 속성값의 경로에서 문서를 가져오는 역할을 한다.

⭐️ <body> 태그 Block Level Element

<body> 태그는 html 문서의 내용을 작성하며, 여러가지 요소들을 구조와 문법에 맞게 작성하면, 웹페이지를 원하는 구조로 작성 할 수 있다.

<!DOCTYPE html>
<html lang="ko">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>
Contents
</body>
</html>

위의 예시와 같이 DOCTYPE 선언 후 html태그가 나오고, head태그와 body태그가 차례로 나오는 구조를 가지게된다.

⭐️ Heading 태그 Block Level Element

제목을 단계별로 나타내며, h1~h6으로 구성되어있다. 문서의 대제목부터 차례로 사용하며, 대제목은 h1 태그를 사용하되, h1 태그는 중복 사용은 피하는 것이 좋다.

<h1> 대제목 </h1>
<h2> 소제목 </h2>
<h3> 소소제목 </h3>
등 제목간의 중요도를 나타낼 때 사용한다.

⭐️ <p> (Paragraphs) 태그 Block Level Element

하나의 문장이나 문단을 설정할 때 사용하는 태그이다.

<p> 안녕하세요? </p>
<p> 정종성입니다. </p>

⭐️ <hr /> 태그 Block Level Element

빈 요소(Empty Element)로 단독으로 사용되며, 주제가 변경되거나 구분을 주고 싶을 때 사용되며, 사용시 아래 예시와 같이 구분선이 생성된다.

<span> 첫번째주제 </span> <hr />
<span> 두번째주제 </span>

⭐️ <blockquote> 태그 Block Level Element

일반적으로 어떠한 문구를 인용하여 가져올 때 사용된다. 즉, 인용문을 가져올 때 사용하는 태그이다.
속성으로는 cite 속성이 있으며, cite 속성값으로는 인용된 정보의 출처(URL)을 입력한다.

<blockquote cite = "인용문의 출처(URL)"> 인용문 </blockquote>

⭐️ <pre> 태그 Block Level Element

서식이 미리 지정된 텍스트를 보여주는 태그이다. html 문서 내의 줄바꿈이나 공백을 특정한 태그와 함께 사용하여 표현해야 하는데, pre 태그 안에서는 작성한 내용대로 화면에 표시한다.

<pre>
pre 태그 안에서 <br>태그가 없이
줄바꿈을 진행해도
줄바꿈이 표현이 된다.
</pre>

⭐️ <br /> 태그 Inline Level Element

빈 요소(Empty Element)로 단독으로 사용되며, 줄바꿈을 하고 싶을 때 사용한다.

<span> 안녕하세요? </span> <br />
<span> 정종성입니다. </span>

⭐️ <ol>,<ul> 태그 Block Level Element

이 두개의 태그는 목록을 나타낼 때 사용한다. 순차적 목록(Ordered list)인 <ol>, 비순차적 목록(Unordered list)인 <ul> 이 있다.
이 두개의 태그 안에는 <li> 태그만이 자식으로 들어갈 수 있다.

<ol>
<li> 첫번째 </li>
<li> 두번째 </li>
<li> 세번째 </li>
</ol>

위와 같이 작성을 하게되면 아래와 같은 결과를 가지게된다.

  1. 첫번째
  2. 두번째
  3. 세번째

<ol> 태그는 순차적인 목록을 작성할 때 사용한다.

<ol type= "a" start = "4" reversed>
<li> 첫번째 </li>
<li> 두번째 </li>
<li> 세번째 </li>
</ol>

위와 같이 작성을 하게되면 아래와 같은 결과를 가지게된다.

  1. 첫번째
  2. 두번째
  3. 세번째

<ol> 태그의 속성인 type 속성에 속성값으로 " 1(기본값), a, A, i, I "가 있으며, 항목에 사용할 번호의 종류 설정한다.
<ol> 태그의 속성인 start 속성에 숫자를 입력하면 항목의 시작 숫자 설정한다.
<ol> 태그의 속성인 reversed는 말 그대로, 순서를 역순으로 설정해서 보여준다.

<ul>
<li> 첫번째 </li>
<li> 두번째 </li>
<li> 세번째 </li>
</ul>

위와 같이 작성을 하게되면 아래와 같은 결과를 가지게된다.

  • 첫번째
  • 두번째
  • 세번째

<ul> 태그는 비순차적인 목록을 작성할 때 사용한다.

<ul type= "square">
<li> 첫번째 </li>
<li> 두번째 </li>
<li> 세번째 </li>
</ul>

위와 같이 작성을 하게되면 아래와 같은 결과를 가지게된다.

  • 첫번째
  • 두번째
  • 세번째

<ul> 태그의 속성인 type 속성에 속성값으로 " disc(기본값), square, circle, none " 이 있으며, 항목에 사용할 기호의 종류 설정한다.

⭐️ <li> 태그 list-item Element

이 태그는 목록의 리스트를 나타낼 때 사용한다. <li> 태그는 단독으로 사용이 불가능하며, 꼭 <ul> 혹은 <ul> 태그를 부모로 두어야 사용이 가능하다.
위의 <ol>, <ul> 의 예시만 보아도 사용법을 알 수 있다.

⭐️ <dl>,<dt>,<dd> 태그 Block Level Element

위의 태그들은 용어에 대한 설명을 지정하여 사용하는 태그이다. <dl> 태그 안에는 <dt>, <dd> 외의 태그는 자식으로 존재가 불가능하다.

  • <dl> : 해당 범위가 용어와 용어에 대한 설명임을 정의한다.

  • <dt> : 용어를 지칭한다.

  • <dd> : 용어에 대한 설명을 일컫는다.

<dl>
<dt> Firefox </dt>
<dd> 모질라 재단에서 만든 웹 브라우저 </dd>
</dl>

위와 같이 작성을 하게되면 아래와 같은 결과를 가지게된다.

Firefox
모질라 재단에서 만든 웹 브라우저

용어의 대한 설명을 지정하여 작성하여준다. 하나의 용어에 여러개의 설명이 있다면, <dd> 태그를 추가하여 작성해주면 된다.
