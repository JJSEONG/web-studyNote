HTML의 form 요소(Element)

⭐️ HTML 양식 만들기

HTML 문서 내에서 어떠한 내용값을 받아서 웹 서버로 보내주는 양식을 만들어주는 요소들이다. 이 요소들을 가지고 입력 받는 양식을 어떻게 만들어나가는지 알아보고, 각 요소들의 속성에 대한 내용도 알아보자.

⭐️ <form> 태그
Block Level Element

<form> 태그(Tag)는 입력받은 내용들을 웹 서버로 보내주는 내용들의 범위를 설정한다. form 태그 안에 또 다른 form 태그는 작성할 수 없다. form 태그와 함께 사용하는 속성들을 알아보자.

<form 속성 = "속성값"> form 태그를 구성하는 여러 요소들 </form>
속성 의미
autocomplete 사용자가 사용하던 내용을 기억하여, 자동으로 문구가 완성되는 기능
속성값으로는 on / off 가 있으며, 기본값은 on 이다.
action = "url" 입력받은 데이터를 처리할 프로그램의 url 지정한다.
name="이름지정" 하나의 문서에 여러개의 폼요소가 있을 경우, 폼요소를 식별할 수 있는 이름 지정한다.
target 서버로 전송 한 뒤, 브라우저의 응답받을 곳을 지정한다.
속성값으로는 _self, _blank 가 존재한다.
method 입력한 정보를 서버에 전달하는 역할이다.
속성값으로는 GET / POST 가 존재한다.
  • GET : 검색창과 같은 간단한 정보 전달
  • POST : 보안유지가 좋아 비밀번호와 같은 중요한 데이터 전송할 때 사용

⭐️ <input /> 태그
Inline-Block Level Element

<input> 태그(Tag)는 빈 태그(Empty Tag)로 브라우저 내에서 사용자에게 어떠한 입력을 받고, 서버로 받아 들이기 위해 사용하는 태그이다. 주로 설문조사나 성별 등을 입력받거나, 검색할 수 있는 칸을 생성할때 사용한다.

<input type = "type의 값" 속성 = "속성값" />
속성 의미
autocomplete 사용자가 사용하던 내용을 기억하여, 자동으로 문구가 완성되는 기능
속성값으로는 on / off 가 있으며, 기본값은 on 이다.
disabled 입력받을 양식을 비활성화하여 사용자가 작성을 불가능하게 만들어준다.
checked type="radio / checkbox" 일 때, 미리 체크가 표시 된 상태로 브라우저에 표시된다.
maxlength="숫자" 작성할 수 있는 최대 글자의 수를 제한한다.
name="이름지정" 양식의 이름을 지정하며, 같은 주제의 내용을 동일 이름으로 지정하여 사용한다.
placeholder="힌트작성" 입력양식에 무엇을 작성해야 하는지, 힌트를 알려줄 수 있는 역할을 한다.
readonly 읽기전용으로, 사용자가 수정이 불가능하게 만들어준다.
type="타입의 종류" 해당 입력양식에 받아 올 데이터의 종류를 얘기한다. type의 종류를 알맞게 사용하여야한다.
value="초기 설정값" 브라우저에서 입력양식을 보았을 때 미리 작성되어있는 값을 얘기한다.
step="숫자" number, range 속성 일 경우 step 속성을 사용하여 증감 폭을 설정해준다.

⭐️ <input /> 태그안의 type 속성의 종류

<input> 태그(Tag)의 속성인 type 에는 여러 데이터 종류가 있다. 이 데이터들을 알맞는 용도로 적재적소에 사용 할 줄 알아야한다.

<input type = "type의 값" />
데이터의 종류 의미
text 일반적인 텍스트를 입력하는 양식을 만들어준다.
password 비밀번호를 입혁할 수 있는 양식을 만들어준다.
radio 라디오 버튼으로 생성되어 단일 선택을 하기 위해 사용되며, <input>태그의 name 속성을 이용하여 그룹화하여 사용한다.
checkbox 체크박스 버튼으로 생성되어 다중 선택을 하기 위해 사용되며, <input>태그의 name 속성을 이용하여 그룹화하여 사용한다.
file 파일을 업로드 할 수 있는 버튼을 생성된다.
button 일반적인 버튼으로 생성되며, <button> 태그와 동일한 역할을 한다.
submit 전송(제출)버튼으로 생성되며, 입력한 내용을 전송(제출)을 위해 사용한다.
reset 초기화 버튼으로 생성되며, 작성한 내용이 초기화되어 작성한 내용을 모두 지우고 재입력을 할 수 있다.
search 검색을 할 수 있는 텍스트 입력창을 생성한다.
tel 전화번호를 입력 할 수 있는 창이며, 모바일에서 클릭 시, 숫자패드가 우선으로 표현된다.
url url을 입력 할 수 있는 텍스트 입력창을 생성한다.
email 이메일을 작성 할 수 있는 텍스트 입력창을 생성한다.
datetime-local 날짜와 시간을 선택할 수 있는 입력창을 생성한다.
date 날짜를 선택할 수 있는 입력창을 생성한다.
month 년,월을 선택할 수 있는 입력창을 생성한다.
week 해당 년도의 몇번째 주인지 선택할 수 있는 입력창을 생성한다.
time 시간을 지정할 수 있는 입력창을 생성한다.
range 범위를 설정할 수 있는 슬라이드 막대가 생성된다.
number 숫자를 입력할 수 있는 입력창을 생성한다. <input>의 속성 step 을 사용하여, 증감 폭을 정할 수 있다.
color 색상을 지정 할 수 있는 선택창을 생성한다.

⭐️ <button> 태그
Inline-Block Level Element

<button> 태그(Tag)는 <input> 태그의 type="button"의 값과 같은 의미를 가진다. type 속성의 값으로는 button[기본값], submit, reset 이 있다.

<button type = "button[기본값] / submit / reset"> Button! <button>
속성 의미
disabled 입력받을 양식을 비활성화하여 사용자가 작성을 불가능하게 만들어준다.
name="이름지정" 데이터와 함께 전송되는 버튼의 이름을 얘기한다.
type="타입의 종류" button[기본값] / submit / reset 이 있으며, 선택하여 사용한다.

⭐️ <textarea> 태그
Inline-Block Level Element

<textarea> 태그(Tag)는 여러 줄을 입력 할 수 있는 입력창을 생성한다. 이용약관 등을 나타낼 때 주로 사용한다.

<textarea 속성 = "속성값"><textarea>
속성 의미
autocomplete 사용자가 사용하던 내용을 기억하여, 자동으로 문구가 완성되는 기능
속성값으로는 on / off 가 있으며, 기본값은 on 이다.
disabled 입력받을 양식을 비활성화하여 사용자가 작성을 불가능하게 만들어준다.
maxlength="숫자" 작성할 수 있는 최대 글자의 수를 제한한다.
name="이름지정" 양식의 이름을 지정한다.
placeholder="힌트작성" 입력양식에 무엇을 작성해야 하는지, 힌트를 알려줄 수 있는 역할을 한다.
readonly 읽기전용으로, 사용자가 수정이 불가능하게 만들어준다.
rows="영역의 줄 수" 표현해 줄 수 있는 줄 수를 설정해주며, 줄 수를 넘어가면 스크롤이 생성된다.
cols="영역의 너비" 표현해 줄 수 있는 너비를 설정해준다.

⭐️ <select> 태그
Inline-Block Level Element

<select> 태그(Tag)는 드롭다운 목록을 생성하며, <option> 태그를 자식으로 두어, 선택을 할 수 있는 선택창을 생성한다.

<select 속성 = "속성값">
<option> 옵션 1 </option>
<option> 옵션 2 </option>
<option> 옵션 3 </option>
</select>
속성 의미
autocomplete 사용자가 사용하던 내용을 기억하여, 자동으로 문구가 완성되는 기능
속성값으로는 on / off 가 있으며, 기본값은 on 이다.
disabled 입력받을 양식을 비활성화하여 사용자가 작성을 불가능하게 만들어준다.
multiple 두개 이상의 항목을 선택할 수 있게 해준다.
name="이름지정" 양식의 이름을 지정한다.
size="숫자" 한번에 볼 수 있는 옵션의 갯수를 설정한다.

⭐️ <datalist> 태그

<datalist> 태그(Tag)는 텍스트를 입력하는 필드에 사전에 미리 입력된 목록을 드롭다운 형식으로 출력하여 보여준다.
<input> 태그의 list 값과 <datalist> 태그의 id 값이 동일하게 작성되어야 한다.

<input type = "text" list = "color" />
<datalist id = "color">
<option> RED </option>
<option> GREEN </option>
<option> BLUE </option>
</datalist>
<input type = "text" list = "color" />
<datalist id = "color">
<option value = "RED">
<option value = "GREEN">
<option value = "BLUE">
</datalist>

위의 두개의 예시처럼 둘 다 정상적으로 출력이 되는 걸 알 수 있다. 아래의 <option> 태그에 value값은 서버로 넘어가는 값이며, label값은 사용자에게 보여지는 값이다. label값이 없다면, value값이 label의 역할을 같이 한다.

속성 의미
autocomplete 사용자가 사용하던 내용을 기억하여, 자동으로 문구가 완성되는 기능
속성값으로는 on / off 가 있으며, 기본값은 on 이다.
disabled 입력받을 양식을 비활성화하여 사용자가 작성을 불가능하게 만들어준다.
multiple 두개 이상의 항목을 선택할 수 있게 해준다.
name="이름지정" 양식의 이름을 지정한다.
size="숫자" 한번에 볼 수 있는 옵션의 갯수를 설정한다.

⭐️ <option> 태그

<option>태그(Tag)는 <select> 태그나 <datalist> 태그와 함께 사용되는 옵션 선택 목록이다.

<select 속성 = "속성값">
<option> 옵션 1 </option>
<option> 옵션 2 </option>
<option> 옵션 3 </option>
</select>
속성 의미
disabled 입력받을 양식을 비활성화하여 사용자가 작성을 불가능하게 만들어준다.
value 서버로 전송되는 데이터의 값이다.
label 사용자가 볼 수 있는 옵션의 제목이다. 생략 시 value값이 대신 보여진다.
selected 옵션의 종류 중 미리 선택이 된 것을 표시해준다.

⭐️ <label> 태그
Inline Level Element

<label> 태그(Tag)는 입력창과 입력창의 제목을 그룹화하여 제목을 클릭하여도 입력창이 자동으로 포커싱 될 수 있게 해주는 태그이다. 사용성과 웹 접근성이 향상되며 <label> 태그를 사용하는 것을 권장한다.
<label> 태그의 속성에는 for 속성이 있으며, 입력창의 id 값과 동일하게 작성 되어야한다.

<label for = "age">나이를 입력하세요</label>
<input type = "text" id = "age">

속성 의미
disabled 입력받을 양식을 비활성화하여 사용자가 작성을 불가능하게 만들어준다.
value 서버로 전송되는 데이터의 값이다.
label 사용자가 볼 수 있는 옵션의 제목이다. 생략 시 value값이 대신 보여진다.
selected 옵션의 종류 중 미리 선택이 된 것을 표시해준다.

⭐️ <fieldset>, <legend> 태그
Block Level Element

<fieldset> 태그(Tag)는 여러개의 입력창들 중에 같은 주제를 가진 내용들을 그룹으로 정하여 범위를 설정한다.
<legend> 태그(Tag)는 <fieldset> 태그로 그룹을 정한 범위를 border로 둘러싸준다.

<fieldset>
<legend>수강과목</legend>
<p>
<label for="time"> 영어회화(초급)</label> <input id="time" type="text" value="오전 9:00~11:00" readonly>
</p>
</fieldset>
