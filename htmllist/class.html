HTML의 class와 id 속성

⭐️ HTML 요소에 이름 할당하기

HTML 문서 내에서 특정한 부분에만 스타일을 지정하고 싶거나, 동일한 부분을 반복적으로 스타일을 주고 싶을 때, 각 태그마다 class와 id 속성을 사용하여 이름을 지정하여 줄 수 있으며, 이를 가지고 스타일을 꾸밀 수 있다. 이 두 태그는 전역속성이며, 존재하는 모든 HTML의 요소에 공통적으로 사용이 가능하다. class와 id 속성의 차이점을 알아보자.

⭐️ class 속성

class 속성은 이름을 지정하고 싶은 태그에 주로 사용된다. HTML의 문서 안에 어느 태그에나 적용이 가능하며, CSS에서 .class명 으로 불러와서 스타일을 줄 수 있다. class속성은 동일한 class명을 가진 class를 여러개 동시에 사용이 가능하다.

⭐️ id 태그

id 속성도 이름을 지정하고 싶은 태그에 주로 사용된다. HTML의 문서 안에 어느 태그에나 적용이 가능하며, CSS에서 #class명 으로 불러와서 스타일을 줄 수 있다. id속성은 단 하나의 id명을 가진 id만 사용 가능하다.


속성 의미
class="class명"
  • HTML문서내의 모든 요소에 사용 가능한 속성이다.
  • CSS에서 불러올 때, .class명 으로 불러온다.
  • 동일한 class명으로 여러 class를 가질 수 있다.(중복사용가능)
  • 동일한 스타일을 반복적으로 적용하기에 적합하다.
  • 하나의 요소에 여러 class속성을 가져도 무관하다.
id="id명"
  • HTML문서내의 모든 요소에 사용 가능한 속성이다.
  • CSS에서 불러올 때, #id명 으로 불러온다.
  • 하나의 id명을 가진 id속성은 고유해야한다.(중복사용불가능)
  • 특정요소에만 개별의 스타일을 줄 때 사용하기 적합하다.
  • 하나의 요소에 하나의 id속성만 가질 수 있다.
<p class = "color"> RED </p>
<p class = "color"> BLUE </p>
<p class = "color"> GREEN </p>

위의 내용처럼 class 속성은 동일한 class명으로 중복 사용이 가능하다.

 <p id = "color1"> RED </p>
<p id = "color2"> BLUE </p>
<p id = "color3"> GREEN </p>

위의 내용처럼 id 속성은 동일한 id명으로 중복 사용이 불가능하며, 여러개의 id명을 가지는 것도 불가능하다.
그로 인해 서로 다른 id명을 부여한 것을 알 수 있다.

 <p class = "color favorite"> RED </p>
<p class = "color hate"> BLUE </p>
<p class = "color favorite"> GREEN </p>

위의 내용처럼 class 속성은 여러개의 class명을 가질 수 있다.

 <p class = "color" id = "favorite"> RED </p>

id 속성은 중복이 불가능하고 고유하여야하지만, id 속성이 아닌 class 속성과는 동시에 같이 사용이 가능하다.
